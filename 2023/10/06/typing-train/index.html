typing_train

2023/10/6

题目要求

把题目文本中出现的字符串复制粘贴提交3000次即可得出flag

解题过程

f12查看页面中字符串在哪里出现，发现要获得标签${

}$中的文本内容，复制在input里
题解脚本如下：

import time
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.common.keys import Keys
from selenium.webdriver.common.action_chains import ActionChains

driver = webdriver.Chrome()
driver.get("http://localhost:52571/index.php?start")
# 循环提交次数


for _ in range(3200):

    time.sleep(0.01)
    
    # 获取<h1 class="typing">标签的文本内容
    h1_element = driver.find_element(By.CSS_SELECTOR, "h1.typing")
    text_to_copy = h1_element.text
    
    # 找到<input>元素并设置文本内容
    input_element = driver.find_element(By.NAME, "input")
    input_element.clear()  # 清空字段内容
    input_element.send_keys(text_to_copy)  # 设置文本内容
    
    # 模拟按回车键以提交表单
    action = ActionChains(driver)
    action.send_keys(Keys.RETURN)
    action.perform()
    
    print(f'Submission {_+1} successful. Text: {text_to_copy}')
input("Press Enter to continue...")
driver.minimize_window()

这个脚本使用了selenium库自动化web浏览器操作示例，具体可查selenium中文文档
注意，在执行完页面操作之后需要添加等待我们手动关闭的操作，不然的话辛苦跑了几分钟的脚本直接自动关闭了还没有flag巨难受（
